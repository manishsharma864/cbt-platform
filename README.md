# CBT Platform Documentation

## Overview
The CBT Platform is designed to provide a comprehensive solution for computer-based testing. It supports the creation, management, and execution of tests and offers a user-friendly interface for both administrators and test-takers.

## Features
- **User Management:** Admins can create and manage user accounts, including students and instructors.
- **Test Creation:** Easily design and configure tests, including multiple-choice and essay questions.
- **Results Tracking:** View and analyze test results for continuous improvement and assessment.
- **Analytics Dashboard:** Get insights into user performance and test effectiveness.

## Installation
To set up the CBT Platform, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/{owner}/cbt-platform.git
   cd cbt-platform
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Configure the environment variables (see `.env.example` for reference).
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- Navigate to the application URL in your browser after starting the server.
- Log in with your admin credentials to create and manage tests.
